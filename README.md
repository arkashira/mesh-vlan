# 🚀 mesh-vlan
**Power network administrators with scalable wifi access point solutions, providing a simple UI for managing VLAN, multiple SSIDs, and mesh networking capabilities.** mesh-vlan is a wifi access point solution designed to overcome signal loss in challenging environments.

## Why mesh-vlan?
- **Scalable**: Supports multiple SSIDs and VLANs, making it ideal for large-scale networks.
- **Easy to use**: Features a simple and intuitive UI for easy management and configuration.
- **Reliable**: Utilizes mesh networking capabilities to ensure a stable and reliable connection.
- **Flexible**: Allows for SSID-to-VLAN mapping, providing flexibility in network configuration.
- **Secure**: Supports secure authentication and encryption protocols to protect network data.
- **Customizable**: Can be customized to meet the specific needs of various environments and use cases.
- **Cost-effective**: Provides a cost-effective solution for wifi access point needs.

## Feature Overview
| Feature | Description |
| --- | --- |
| Multiple SSIDs | Supports multiple SSIDs for separate network configurations |
| VLAN Support | Allows for VLAN configuration and management |
| Mesh Networking | Utilizes mesh networking capabilities for reliable connections |
| SSID-to-VLAN Mapping | Enables mapping of SSIDs to specific VLANs |
| Simple UI | Features a simple and intuitive UI for easy management and configuration |
